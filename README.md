## Install and Run

1. Go, clone repo

2. Install Postgres v15 or later:

```bash
sudo apt update
sudo apt install postgresql postgresql-contrib
```

To check current version:

```bash
psql --version
```

3. (Linux only) Update postgres password:

```bash
sudo passwd postgres
```
